Question/Answers

1. What are Section, Header, Footer tags? Semantic tags?

Semantic tags are HTML tags that describe their meaning. <header> = top of page, <section> = part of page, <footer> = bottom of page.

They are used to make HTML easy to understand, improve SEO, and help screen readers.

2. What is Mobile Friendly or Mobile First?

Mobile friendly means the website works properly on phones. Mobile First means designing for mobile screens first, then bigger screens.

3. What HTML/CSS/JS makes a website mobile friendly?

✔ Meta viewport tag ✔ CSS media queries ✔ Responsive units (% / rem / vw) ✔ Flexbox or Grid layouts ✔ JavaScript for responsive menus (optional)

4. What is Inline Style?

Inline style means CSS written directly inside an HTML tag:
<p style="color:red;">Hello</p>
